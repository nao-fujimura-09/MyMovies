## 背景
  
映画鑑賞が趣味な方はまとめて何本か見ることが多いと思います。  
閲覧した映画のデータや、気になる映画の情報を手軽に管理できればと思い、このサイトを作成しました。  


また、同じジャンルの趣味を持つユーザーをフォローすることにより、どの映画を見ようか悩んだ時に参考にする事が可能です。  
<br><br>

## 使用方法

ログイン後、映画一覧画面にて各ボタンを押下することで、下記の動作をまとめて行うことが可能。  

* 星評価
* ウォッチリストへの追加
* 閲覧の有無


◎ 度々詳細画面を開かなくてよいので手軽に評価や管理ができる。  
◎ 星評価のみのレビューが可能なので時間がない時でも簡単に評価ができる。

## 実装機能

### User（利用者）

|  機能 |  説明  |非ログイン時<br>利用可否 |
| ---- | ---- |:------------------:|
|新規登録機能|ユーザーネーム・メールアドレス・パスワードで新規登録可能|○|
|ログイン機能|ユーザーネーム・メールアドレス・パスワードでログイン可能|○|
|ログアウト機能|ヘッダーの LOG OUT ボタンを押下するとログアウトが可能|×|
|退会機能|マイページのプローフィール画面の退会ボタンを押下すると退会が可能|×|
|映画作品一覧　閲覧機能|トップページの人気の作品・評価順の作品情報を閲覧できる<br>ログインユーザーはジャンル別やタイトルで検索できる|○<br>×|
|映画作品詳細　閲覧機能|作品のタイトルを押下すると作品の詳細を閲覧売ることができる|○|
|レビュー機能<br>（タイトル、本文、星評価）|作品詳細画面からレビューが可能（タイトル・本本・星評価）<br>トップページ・映画作品一覧画面・映画作品詳細画面で星評価が可能|×|
|レビュー編集機能<br>（タイトル、本文、星評価）|作品詳細画面からレビューの編集が可能<br>（タイトル・本本・星評価）|×|
|レビュー削除機能|作品詳細画面からレビューの削除が可能|×|
|ウォッチリストへの追加・削除|トップページ・映画作品一覧画面・映画作品詳細画面でウォッチリストへの<br>追加・削除が可能<br>(ウィッチリスト一覧画面に表示される)   |×|
|閲覧の有無|トップページ・映画作品一覧画面・映画作品詳細画面で閲覧済み・未閲覧の<br>選択が可能<br>（マイページに閲覧済み作品が表示される）|×|
|フォロー機能|トップページの人気のユーザー・映画作品詳細画面のユーザーの<br>名前を押下すると、フォロー・フォローの解除が可能|×|

<br><br>

### Admin(管理者）

|  機能 |  説明  |非ログイン時<br>利用可否 |
| ---- | ---- |:------------------:|
|ログイン機能|ユーザーネーム・メールアドレス・パスワードでログイン可能|○|
|映画作品のデータ閲覧機能|作品ごとの情報やデータを閲覧することが可能<br>（閲覧数・平均評価・レビュー数・ウォッチリストへの追加数）<br>上記の項目で並び替えすることが可能|×|
|映画作品詳細データ閲覧機能|タイトルを押下すると映画作品詳細画面を閲覧することが可能|×|
|ジャンル一覧　閲覧機能機能|ジャンルの一覧を閲覧することが可能|×|
|ユーザーデータ閲覧・編集機能|ユーザーごとの情報やデータを閲覧することが可能<br>基本情報：ID・ユーザーネーム・メールアドレス・ステータス<br>フォロー状況：フォロワー数・フォロー数<br>分析データ：映画閲覧数・レビュー数・ウォッチリストへの追加数・平均評価）|×|
